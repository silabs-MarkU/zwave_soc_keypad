button

Emulating button events (various type of button presses).
button press u8 u8 button_press_from_cli
Emulating a button press
Alias btn p
Arguments
  • u8Button: 0: BTN0, 1+: BTN1
  • u8Press and hold duration: 0 - short, 1- medium, 2 - long, 3+ - verylong
button hold u8 button_hold_from_cli
Emulates pressing and holding a button.
Alias btn h
Arguments
  • u8Button: 0: BTN0, 1+: BTN1
button release u8 button_release_from_cli
Emulates releasing a button.
Alias btn r
Arguments
  • u8Button: 0: BTN0, 1+: BTN1
